Setting up HoloLens 2 for Development

Most of the tutorial follows this tutorial in allowing the user to deploy apps to the HoloLens.

I'd highly recommend that you follow that tutorial if you have any trouble.

Allowing developer access

  1. Open settings from the start menu
  2. Select Update
  3. Select For developers
  4. Enable Use developer features and Device discovery
  5. There is an setting in there for pairing, keep in mind this location for later when you are setting up a connection to your PC
