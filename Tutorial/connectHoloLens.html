Connecting to the HoloLens from your computer

Make sure that the HoloLens is connected to the same network as your computer.

Next, you want to get the IP address of the HoloLens.

Open Settings->Network and Internet->[Wifi name]->Advanced Options

Scroll down until you can find the IPV4 address and remember it

You can also enter the IP address below for reference

Connecting to the HoloLens via the Microsoft HoloLens app

The Microsoft HoloLens app is able to allow you to screenshare what you are seeing with the HoloLens directly to your computer and is also able to take photos/videos, view the files stored in the HoloLens, and remote control the HoloLens.

You can download it to your computer here.

In order to connect it to your computer, click on the "add" button and type in .

It will now prompt you to create a new username and password to connect with the HoloLens. Create one on your computer at this moment.

You should now be connected with the HoloLens via the HoloLens app.

Connecting to the HoloLens via Visual Studio

  1. Right below the menu bar, you should see the debug toolbar. Change the debug toolbar items to be Debug, ARM, and whatever app you want to run.
  2. Next, on the button with the green arrow, select Remote Machine.
  3. If a menu bar popped up, then proceed to the next step. Otherwise, right click the app you want to deploy to the HoloLens and then click on properties->debug
  4. In Start options, make sure that the target device is remote machine and enter into remote machine. Next, make the authentication mode universal (unencrypted protocol).
  5. Click on the green arrow to start deploying the application to the HoloLens.
